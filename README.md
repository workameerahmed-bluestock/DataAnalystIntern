# Mutual Fund Analytics

## Overview
This project was completed as part of the BlueStock Data Analyst Capstone Program.

The project focuses on cleaning, validating, storing, and analyzing mutual fund data using Python and SQL.

## Tools & Technologies
- Python
- Pandas
- SQLite
- SQL
- Git & GitHub
- Matplotlib
- Seaborn
- Plotly

## Features
- Data Cleaning & Validation
- SQLite Database Creation
- SQL Schema Design
- Data Ingestion
- Analytical SQL Queries
- Exploratory Data Analysis (EDA)
- Data Dictionary
